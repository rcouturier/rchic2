RCHIC2 Analyse Statistique Implicative

Guide d'utilisation de l'application RCHIC2, outil d'Analyse Statistique Implicative (ASI) pour l'etude des relations entre variables binaires ou categorielles.

1. Demarrage rapide

  1. Lancez RCHIC2.
  2. Cliquez sur Charger un fichier CSV et selectionnez votre fichier de donnees.
  3. Verifiez que vos variables apparaissent dans le panneau de droite.
  4. Choisissez un type d'analyse : Graphe Implicatif, Arbre de Similarite ou Arbre Hierarchique.
  5. Reglez les options si necessaire, puis cliquez sur Calculer.
  6. Explorez le resultat dans l'onglet Visualisation et les details dans l'onglet Donnees de calcul.

2. Preparer ses donnees

RCHIC2 accepte des fichiers au format CSV (valeurs separees par des points-virgules).

Structure attendue

Variable1Variable2Taille.pSexe.s
Eleve1101651
Eleve2011420
  • Premiere colonne : identifiants des observations (eleves, individus...).
  • Colonnes suivantes : variables binaires (0/1) ou numeriques.

Suffixes speciaux

SuffixeSignificationExemple
.pVariable a partitionner : variable numerique continue qui sera automatiquement decoupee en 3 classes (par defaut) via un algorithme de classification.Taille.p
.sVariable supplementaire : exclue de l'analyse principale. Utilisee uniquement pour le calcul des contributions et typicalites.Sexe.s
(aucun)Variable d'analyse standard (binaire 0/1).ReussiteA
Conseil : Les variables supplementaires (.s) permettent de caracteriser les classes ou chemins trouves par l'analyse, par exemple pour identifier si un groupe est plutot compose de filles ou de garcons.

3. Interface de l'application

Barre laterale (gauche)

  • Charger un fichier CSV : ouvre le selecteur de fichier.
  • Informations : nom du fichier, nombre d'observations et de variables.
  • Calculer : lance l'analyse selectionnee.
  • Exporter : sauvegarde l'image et les donnees.
  • Type d'analyse : choix entre les trois types d'analyse.
  • Options : parametres specifiques a chaque type d'analyse.

Panneau des variables (droite)

  • Liste de toutes les variables avec cases a cocher.
  • Boutons Tout / Aucun pour selectionner ou deselectionner rapidement.
  • Seules les variables cochees sont incluses dans le calcul.

Zone principale

  • Onglet Visualisation : affiche le graphe ou l'arbre.
  • Onglet Donnees de calcul : affiche la console avec les resultats detailles.

Selecteur de langue

En haut a droite, choisissez entre FR (francais), EN (anglais) ou PT (portugais). La langue affecte l'interface, les messages de la console et le format d'export CSV.

4. Graphe Implicatif

Le graphe implicatif represente les relations d'implication statistique entre les variables. Un arc A → B signifie : "quand A est vrai, B tend a etre vrai aussi".

Modes de calcul

ModeDescription
ClassiqueIndice d'implication standard.
Classique + ConfianceCombine l'indice d'implication et la confiance de la regle.
ImplifianceIndice d'implifiance, specifique a l'ASI.
EntropiqueIndice base sur l'entropie (theorie de l'information).

Seuils et couleurs

Quatre seuils independants (de 50 a 100) controlent l'affichage des arcs :

  • Chaque seuil a une case d'activation, un curseur et un choix de couleur.
  • Un arc est affiche si sa valeur d'implication depasse au moins un seuil actif.
  • La couleur de l'arc correspond au seuil le plus eleve qu'il depasse.

Options supplementaires

  • Graphe complet : affiche les implications dans les deux sens (A→B et B→A).
  • Afficher la confiance : inscrit la valeur de confiance sur chaque arc.

Interagir avec le graphe

  • Clic sur un arc : affiche les details de la regle (implication, confiance, contre-exemples).
  • Clic sur un noeud : affiche le nom de la variable et ses occurrences.
  • Clic droit sur un noeud : calcule les contributions et/ou typicalites des variables supplementaires a partir de ce noeud (voir section 7).

5. Arbre de Similarite

L'arbre de similarite regroupe les variables selon leur degre de ressemblance. Plus deux variables sont similaires dans leurs reponses, plus elles sont proches dans l'arbre.

Options

  • Afficher les similarites : inscrit la valeur de similarite (0-1) a chaque niveau de fusion.
  • Contributions / Typicalites : active le calcul avec les variables supplementaires (voir section 7).

Lecture de l'arbre

  • Les variables sont affichees en haut (feuilles).
  • Les lignes horizontales indiquent les fusions entre groupes.
  • Les fusions en rouge sont statistiquement significatives.
  • Les valeurs de similarite apparaissent au-dessus de chaque fusion.

6. Arbre Hierarchique

L'arbre hierarchique regroupe les variables en classes orientees, en tenant compte du sens de l'implication. La cohesion mesure la force du lien au sein de chaque classe.

Modes de calcul

ModeDescription
ClassiqueCohesion standard.
Classique + ConfianceCohesion ponderee par la confiance.
ImplifianceCohesion basee sur l'implifiance.

Options

  • Afficher les cohesions : inscrit la valeur de cohesion a chaque niveau.
  • Contributions / Typicalites : active le calcul avec les variables supplementaires (voir section 7).

7. Contributions et Typicalites

Ces analyses permettent de caracteriser les classes ou chemins identifies par l'analyse a l'aide de variables supplementaires (colonnes avec le suffixe .s).

Prerequis : Votre fichier CSV doit contenir au moins une variable supplementaire (suffixe .s) pour que ces calculs soient disponibles.

Contribution

Mesure a quel point chaque variable supplementaire contribue a la formation d'une classe ou d'un chemin. Un risque faible indique une contribution forte.

Typicalite

Mesure a quel point chaque variable supplementaire est representative (typique) d'une classe ou d'un chemin. Un risque faible indique une forte typicalite.

Comment les activer

Type d'analyseMethode
Graphe ImplicatifClic droit sur un noeud → menu contextuel avec 3 choix : Contributions, Typicalites, ou les deux.
Arbre de SimilariteCocher Contributions et/ou Typicalites dans la section "Variables supplementaires" des options, avant de calculer.
Arbre HierarchiqueIdem que pour l'arbre de similarite.

Lecture des resultats

Les resultats apparaissent dans l'onglet Donnees de calcul. Pour chaque classe ou chemin :

  • Chaque variable supplementaire est listee avec son risque (probabilite).
  • La variable avec le risque le plus faible est mise en evidence (>>).
  • Un risque < 0.05 est generalement considere comme significatif.

8. Exporter les resultats

Le bouton Exporter sauvegarde a la fois l'image et les donnees :

AnalyseImageDonnees
Graphe ImplicatifPNG (rchic_graph.png)Matrice d'implication (CSV)
Arbre de SimilariteSVG (rchic_tree.svg)Matrice de similarite (CSV)
Arbre HierarchiqueSVG (rchic_tree.svg)Matrice de cohesion (CSV)
Format CSV : Le format du fichier CSV depend de la langue selectionnee. En francais et portugais, le separateur est le point-virgule (;) et la virgule est le separateur decimal. En anglais, c'est la virgule (,) et le point decimal.

9. Console de calcul

L'onglet Donnees de calcul affiche l'ensemble des resultats numeriques :

  • Parametres du calcul (mode, seuil, options activees).
  • Nombre de variables, regles, noeuds, aretes.
  • Detail des classifications (pour les arbres).
  • Contributions et typicalites (si activees).
  • Tableau des regles (pour le graphe implicatif).

Utilisez Actualiser pour recharger et Effacer pour vider la console.
